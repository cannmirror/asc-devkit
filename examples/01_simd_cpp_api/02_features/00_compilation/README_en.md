# Compilation Sample Introduction

## Overview

This sample introduces the implementation methods for custom operator build projects and static Aclnn calls.

## Sample List

| Directory Name | Description |
| -------------- | ----------- |
| [custom_op](./custom_op/) | This sample introduces the implementation method for custom operator build projects, including Add and LeakyRelu operators |
| [custom_op_static_lib](./custom_op_static_lib/) | This sample introduces the implementation method for static Aclnn calls |
| [parallel_ops_package](./parallel_ops_package/) | This sample introduces the implementation method for multi-vendor parallel build projects for custom operators, including Add and LeakyRelu operators |
