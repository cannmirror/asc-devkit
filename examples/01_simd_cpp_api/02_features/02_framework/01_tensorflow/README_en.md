# TensorFlow Sample Introduction

## Overview

This sample demonstrates the implementation methods for custom operators in the TensorFlow framework.

## Sample List

| Directory Name | Description |
| -------------- | ----------- |
| [tensorflow_builtin](./tensorflow_builtin/) | This sample demonstrates mapping Ascend C custom operators to TensorFlow built-in operators and invoking them through TensorFlow |
| [tensorflow_custom](./tensorflow_custom/) | This sample demonstrates mapping Ascend C custom operators to TensorFlow custom operators and invoking them through TensorFlow |
