# Sorting Operation Sample Introduction

## Overview

This sample set introduces typical usage of different features of sorting operation operators and provides corresponding end-to-end implementations.

## Sample List

| Directory Name               |  Description                                             |
|--------------------| --------------------------------------------------- |
| [sort](./sort)     | This sample is based on the Kernel direct-call operator project. It introduces how to call the Sort high-level API to implement a sort operator, which sorts values in descending order |
| [topk](./topk)     | This sample is based on the Kernel direct-call operator project. It introduces how to call the TopK high-level API to implement a topk operator, which obtains the top k maximum or minimum values and their corresponding indices in the last dimension |
