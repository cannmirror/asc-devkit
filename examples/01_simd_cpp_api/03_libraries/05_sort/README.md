# 排序操作算子样例介绍
 
## 概述
 
本样例集介绍了排序操作算子不同特性的典型用法，给出了对应的端到端实现。
 
## 样例列表
 
| 目录名称               |  功能描述                                             |
|--------------------| --------------------------------------------------- |
| [sort](./sort)     | 本样例基于Kernel直调算子工程，介绍了调用Sort高阶API实现sort单算子，排序函数，按照数值大小进行降序排序 |
| [topk](./topk)     | 本样例基于Kernel直调算子工程，介绍了调用TopK高阶API实现topk单算子，获取最后一个维度的前k个最大值或最小值及其对应的索引 |
