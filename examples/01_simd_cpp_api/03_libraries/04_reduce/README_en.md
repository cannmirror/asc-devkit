# Reduce Operator Example Introduction
 
## Overview
 
This example set introduces typical usage of different features of reduce operators and provides corresponding end-to-end implementations.
 
## Example List
 
| Directory Name                                                                                                                 | Description                                                                        | 
|----------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------| 
| [reduce](./reduce)                                                                                                   | This example demonstrates how to call the WholeReduceSum and BlockReduceSum high-level APIs to implement the reduce operator, achieving cumulative sum of data elements on continuous memory and returning the cumulative result         |
| [reducemax](./reducemax)                                                                                           | This example demonstrates how to compute the maximum value and its corresponding index along a specified dimension for a multi-dimensional vector based on the ReduceMax high-level API.                 |
| [sum](./sum)                                                                                                         | This example demonstrates how to call the Sum high-level API to implement the sum operator, obtaining the sum of elements in the last dimension of the input data                                   |
