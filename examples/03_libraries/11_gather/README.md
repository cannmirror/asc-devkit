# Gather API样例介绍
## 概述
本样例集介绍了Gather API不同特性的典型用法，给出了对应的端到端实现。

## 算子开发样例
| 目录名称                         | 功能描述                                                                                                                                                         |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [gather](./gather)         | 本样例基于Gather实现对于给定的输入张量和一个地址偏移量，根据偏移地址将输入张量按元素收集到结果张量中 |
| [gatherb](./gatherb) | 本样例基于Gatherb实现对于给定的输入张量和一个地址偏移量，根据偏移地址按照DataBlock的粒度将输入张量按元素收集到结果张量中     |
