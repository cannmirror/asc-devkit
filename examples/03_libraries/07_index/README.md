# 索引计算算子样例介绍

## 概述

本样例集介绍了索引计算算子不同特性的典型用法，给出了对应的端到端实现。

## 算子开发样例

|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [arange](./arange) | 本样例基于Kernel直调算子工程，介绍了调用Arange高阶API实现arange单算子，给定起始值，等差值和长度，返回一个等差数列 |
| [create_vec_index](./create_vec_index) | 本样例基于CreateVecIndex实现创建指定起始值的向量索引的方法 |
