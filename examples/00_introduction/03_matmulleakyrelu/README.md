# MatmulLeakyRelu 样例介绍

## 概述

本样例介绍了基于Ascend C的 MatmulLeakyRelu 算子的核函数直调方法，实现计算步骤在硬件层面的高效协同执行，降低内存访问开销与计算延时，可最大化利用AI处理器的并行计算能力。
