# Matmul样例介绍
## 概述
样例介绍了基于Ascend C的Matmul算子的核函数直调方法，可最大化利用AI处理器的并行计算能力，显著提升算子的执行效率，使用与高性能推理与训练场景。

## 算子开发样例
|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [normal_matmul](./normal_matmul) | 本样例介绍Matmul算子的核函数直调方法 |
| [tensor_tile_matmul](./tensor_tile_matmul) | 本样例介绍TensorTile端到端用例的核函数直调方法，算子支持单核运行 |
