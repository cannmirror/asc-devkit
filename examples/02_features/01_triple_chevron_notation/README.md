# PyTorch样例介绍
## 概述
本样例展示了如何使用Pybind与torch.library注册自定义算子，并通过<<<>>>内核调用符调用核函数。

## 算子开发样例
|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [pybind](./pybind) | 本样例展示了如何使用pybind注册自定义算子，并通过<<<>>>内核调用符调用核函数，以简单的Add算子为例，实现两个向量的逐元素相加。 |
| [torch_library](./torch_library) | 本样例展示了如何使用PyTorch的torch.library机制注册自定义算子，并通过<<<>>>内核调用符调用核函数，以简单的Add算子为例，实现两个向量的逐元素相加。 |
