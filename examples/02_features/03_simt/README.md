# Simt样例介绍

## 概述
SIMT新同构场景下的算子实现样例。

## 算子开发样例

|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [simt_gather](./simt_gather)                              | 本样例以Gather算子为示例，展示了在新同构模式下实现一个SIMT算子 |
| [simt_gather_and_simd_adds](./simt_gather_and_simd_adds)  | 本样例以gather和adds算子为示例，展示了在新同构模式下实现SIMT和SIMD混合的算子 |
