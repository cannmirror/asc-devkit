# 协作组简介

协作组（cooperative_groups）是Ascend C SIMT编程模型的一种拓展，用于组织协作线程组。通过协作组能够更细粒度地控制线程协作，从而实现更高效的线程并行协作。

> [!NOTE]说明 
>所有协作组API均在`cooperative_groups`命名空间下，调用时需添加命名空间。

当前提供协作组类型有:

- [thread_block](thread_block/thread_block简介.md)：显示地表达线程块概念。
- [coalesced_group](coalesced_group/coalesced_group简介.md)：Warp中的活跃线程。
- [thread_block_tile](thread_block_tile/thread_block_tile简介.md)：手动切分出的更小协作组。

基于以上协作组类型，提供以下划分操作，支持将一个父组划分为多个子组：

- [tiled_partition](tiled_partition.md)：将父组划分成指定大小的子组。
- [binary_partition](binary_partition.md)：根据标签划分子组。
