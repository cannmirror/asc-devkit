## 函数原型<a name="section620mcpsimp"></a>

```
template<template T>
__aicore__ inline void SetFixpipePreQuantFlag(uint64_t config)
```

float tmp = (float)0.5;

uint64_t deqScalar = static_cast<uint64_t>(*reinterpret_cast<int32_t*>(&tmp));
