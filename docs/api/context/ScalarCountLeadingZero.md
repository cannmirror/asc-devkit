## 功能说明<a name="section618mcpsimp"></a>

计算一个uint64\_t类型数字前导0的个数（二进制从最高位到第一个1一共有多少个0）。

## 函数原型<a name="section620mcpsimp"></a>

```
__aicore__ inline int64_t ScalarCountLeadingZero(uint64_t valueIn)
```
