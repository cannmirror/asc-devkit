## 功能说明<a name="section618mcpsimp"></a>

设置HF32模式取整的具体方式，需要先使用[SetHF32Mode](SetHF32Mode.md)开启HF32取整模式。

## 函数原型<a name="section620mcpsimp"></a>

```
__aicore__ inline void SetHF32TransMode(HF32TransMode mode)
```
