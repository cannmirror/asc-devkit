## 功能说明<a name="section618mcpsimp"></a>

获取输入数据各元素的三角函数余弦值。

![](figures/zh-cn_formulaimage_0000002513329620.png)

## 函数原型<a name="section620mcpsimp"></a>

```
__simt_callee__ inline half2 h2cos(half2 x)
```

```
__simt_callee__ inline bfloat16x2_t h2cos(bfloat16x2_t x)
```

## 返回值说明<a name="section640mcpsimp"></a>

输入数据各元素的三角函数余弦值。

-   当输入元素为inf时，返回值为nan。
-   当输入元素为-inf时，返回值为nan。
-   当输入元素为nan时，返回值为nan。

## 约束说明<a name="section633mcpsimp"></a>

使用本接口时，线程配置最大不超过1024，否则有栈溢出风险。

## 需要包含的头文件<a name="section10354115115916"></a>

使用half2类型接口需要包含"simt\_api/asc\_fp16.h"头文件，使用bfloat16x2\_t类型接口需要包含"simt\_api/asc\_bf16.h"头文件。

```
#include "simt_api/asc_fp16.h"
```

```
#include "simt_api/asc_bf16.h"
```

## 调用示例<a name="section134121647154719"></a>

```
__simt_vf__ __launch_bounds__(1024) inline void KernelCos(__gm__ half2* dst, __gm__ half2* x)
{
    int idx = threadIdx.x + blockIdx.x * blockDim.x;
    dst[idx] = h2cos(x[idx]); // 对src源地址的第idx个元素取三角函数余弦值
}
```
