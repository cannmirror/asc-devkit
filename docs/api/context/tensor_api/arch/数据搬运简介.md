# 数据搬运简介

数据搬运功能支持连续与非连续搬运，搬运时数据排布格式可以保持不变，也可以实现NZ Layout和 RowMajor Layout之间的转换。

下文示例中，每格包含16 \* 32Byte的数据：

**图 1**  非连续搬运，数据排布格式不变（NZ Layout -\> NZ Layout）
![](../figures/非连续搬运-数据排布格式不变（NZ-Layout---NZ-Layout）.png "非连续搬运-数据排布格式不变（NZ-Layout---NZ-Layout）")

**图 2**  连续搬运，数据排布格式改变（RowMajor Layout -\> NZ Layout）
![](../figures/连续搬运-数据排布格式改变（RowMajor-Layout---NZ-Layout）.png "连续搬运-数据排布格式改变（RowMajor-Layout---NZ-Layout）")
