# 运算符重载

## 产品支持情况

| 产品     | 是否支持 |
| ----------- |:----:|
|Ascend 950PR/Ascend 950DT|√|

## 功能说明

重载运算符\(\)，内部实现为调用[Crd2Idx](Crd2Idx.md)接口，将多维坐标（Coordinate）通过布局（Layout）转换为内存位置索引（Index）。

## 函数原型

```
template <typename CoordType>
__aicore__ inline constexpr auto operator()(const CoordType& coord) const {}
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
|--------|-----------|------|
| shape | 输入 | Std::tuple结构类型，用于定义数据的逻辑形状，例如二维矩阵的行数和列数或多维张量的各维度大小。 |
| stride | 输入 | Std::tuple结构类型，用于定义各维度在内存中的步长，即同维度相邻元素在内存中的间隔，间隔的单位为元素，与Shape的维度信息一一对应。 |

## 返回值说明

返回根据Coordinate信息转换之后的索引值。

## 约束说明

无

## 调用示例

```
...
auto layout = AscendC::MakeLayout(shape, stride);
auto index = layout(coord);
```
