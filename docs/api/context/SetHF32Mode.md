## 功能说明<a name="section618mcpsimp"></a>

用于设置Mmad计算是否开启HF32模式，开启该模式后L0A/L0B中的FP32数据将在参与Mmad计算之前被舍入为HF32。

## 函数原型<a name="section620mcpsimp"></a>

```
__aicore__ inline void SetHF32Mode(bool hf32Mode)
```
