## 功能说明<a name="section1695933115912"></a>

设置Mmad计算时优先通过N方向，CUBE将首先通过N方向，然后通过M方向产生结果。

## 函数原型<a name="section620mcpsimp"></a>

```
__aicore__ inline void SetMMColumnMajor()
```

无
