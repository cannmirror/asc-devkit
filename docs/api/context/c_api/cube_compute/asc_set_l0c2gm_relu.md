## 功能说明

数据搬运过程中进行随路量化时，通过调用该接口设置量化流程中的矢量量化参数。

## 函数原型

```c++
__aicore__ inline void asc_set_l0c2gm_relu(uint64_t relu)
```

## 参数说明

|参数名|输入/输出|描述|
|------------|------------|-----------|
| reluPre | 输入     | ReLU激活函数前张量的矢量起始地址。预留参数，暂未启用，为后续的功能扩展做保留，传入一个空地址即可。|
## 返回值说明

无
