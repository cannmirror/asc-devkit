## 约束说明<a name="section633mcpsimp"></a>

如果GetValue的Global Memory地址内容存在被外部改写的可能，需要先调用[DataCacheCleanAndInvalid](DataCacheCleanAndInvalid.md)，确保Data Cache与Global Memory的Cache一致性，之后再调用此接口。
