## 功能说明<a name="section618mcpsimp"></a>

用于阻塞后续的指令执行，直到所有之前的内存访问指令（需要等待的内存位置可通过参数控制）执行结束。

## 函数原型<a name="section620mcpsimp"></a>

```
template <MemDsbT arg0>
__aicore__ inline void DataSyncBarrier()
```
