## 功能说明<a name="section618mcpsimp"></a>

返回指定数据类型的正无穷大值。

Ascend 950PR/Ascend 950DT，支持的数据类型为：half/bfloat16\_t/float。

## 函数原型<a name="section620mcpsimp"></a>

-   标量接口，返回值为标量

    ```
    constexpr __aicore__ static inline T Infinity()
    ```

-   矢量接口，为dstLocal前count个元素赋正无穷大值

    ```
    __aicore__ static inline void Infinity(const LocalTensor<T>& dstLocal, uint32_t count)
    ```

## 返回值说明<a name="section151875718241"></a>

标量接口返回值为对应数据类型的正无穷大值。

矢量接口无返回值。

## 约束说明<a name="section122765452518"></a>

无。

## 调用示例<a name="section16400072257"></a>

-   标量接口

    ```
    float value = AscendC::NumericLimits<float>::Infinity();
    ```

-   矢量接口

    ```
    AscendC::NumericLimits<float>::Infinity(dstLocal, 256);
    ```
