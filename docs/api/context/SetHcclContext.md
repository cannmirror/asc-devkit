## 功能说明<a name="section618mcpsimp"></a>

设置通算融合算子每个通信域对应的context（消息区）地址。

## 函数原型<a name="section620mcpsimp"></a>

```
template <uint32_t index>
__aicore__ inline void SetHcclContext(__gm__ uint8_t* context)
```
