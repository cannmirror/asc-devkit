# 简介<a name="ZH-CN_TOPIC_0000002554206779"></a>

AI Core是AI处理器的计算核心，AI处理器通过多个AI Core实现并行计算。与传统CPU相比，AI处理器更适合模型训练和推理，这得益于其内部更多的计算单元和相应的向量计算指令，使得单个硬件指令能够完成多组数据的计算。AI处理器提供了以下两种编程模型：

-   SIMT（Single Instruction Multiple Thread）：单指令多线程。以单条指令多个线程的形式来实现并行计算。
-   SIMD（Single Instruction Multiple Data）：单指令多数据。以单条指令多个数据的形式来实现并行计算。

SIMT编程适合处理离散访问场景、复杂分支控制场景下的矢量算子开发，也适合熟悉业界SIMT算子开发的人员快速上手Ascend C算子开发。SIMD编程则适用于纯矩阵计算、连续计算的矢量算子或融合算子场景。此外，我们还提供了SIMD与SIMT混合编程的方式，以利用两者的优点实现更优的性能和易用性。
